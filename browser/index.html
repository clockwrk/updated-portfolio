  • Client: Bumble And Bumble

NIGEREAN CROP DASHBOARD

Built this project to show off my backend skills. Here I downloaded CSV dataset files on Nigerian crop prices from http://nigeria.opendataforafrica.org/. Using a a NPM module called baby-parse, I parsed the files, organized and stored the data in a PostgresSQL database. Then I made the data available to the front-end through a custom API. The front-end is then able to use the data to get insight on crop price trends.

  • Date: May 2017
  • Main Technologies: JQuery, HTML5, CSS, JavaScript, NodeJS, PostgresSQL, baby-parse
    • Live Site: DEMO

Imvue

Here I build a andriod mobile web project with my classmates from Fullstack Academy of Code. Users are able to draw and create art projects. Save the project to a specific geolocation along with the tilt and angle. Later another user can come to the same spot and have the art project rendered when aiming their device at the correct tilt, angle and geolocation.

  • Client: Fullstack Academy of Code
  • Date: October 2016
  • Main Technologies: JQuery, HTML5, CSS, JavaScript, NodeJS, PostgresSQL, HTML5 Canvas API
  • Live Site: DEMO
